# YouTube to MP3 - Web Application

Una aplicación web para descargar videos de YouTube en formato MP3.

## Instalación Rápida

### Backend
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
pnpm install
pnpm dev
```

## Requisitos
- Python 3.9+
- Node.js 16+
- FFmpeg

Visita http://localhost:5173 en tu navegador.
